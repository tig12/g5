Results

Here is a description of the characteristics of the downloadable files
Downloadable files are currently in an intermediate state : some corrections are correctly integrated, but some corrections are missing, and many characteristics should be carefully checked before considering them as ready for a reliable use.

Most files are the result of step "2 - cura to csv", except for file A1, which integrates the corrections done during the restoration of 1955 group "570 sportifs"

This page lists the things than need to be bettered, and does not list the things that are already done (date formatting etc.).

Serie A

Notable advances can still be done :
  • Use reverse geocoding to compute timezone offset and restore original birth time, as written in the birth certificate.
  • Compute given name and family name for many records that contain only two strings separated by a whitespace.

File D6

  • Restoration of birth cities with reverse geocoding is not satisfying due to the lack of precision of longitudes and latitudes.

File D10

  • Association of birth cities to geonames.org id needs to be done. Maybe possible by program as birth cities are given. This is a bit problematic because cura file contains approximate spelling of city names, but use of approximate string matching (like Levenstein) may be feasible.

Files E1 E3

  • Misses association of birth cities to geonames.org id needs to be done. The problem is similar to D10.
  • Precise given and family names can be tried even for strings that contain more than one whitespace, using the fact that given names are uppercased in cura file.
