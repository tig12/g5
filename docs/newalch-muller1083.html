This file has a particular historical interset, as it was used for the first replication of a Gauquelin planetary effect completely independantly from Gauquelin.
An other interest of this file is the quality of its data. In particular, person and place names are generally more precise than in file A2.
It contains records from Cura files A2 and E3.

Integration to gauquelin5

Very few work has been done on this file for the moment.
It could be used to
  • Check legal time restoration in A2.
  • Better place names in A2 (then better matching to geonames).
  • Better names in A2
  • And it brings new persons not present in Cura files.

Generate csv

The first step is to generate the file 5-newalch-csv/1083MED.csv : 
php run-g5.php newalch ertel4391 raw2csv

Fix cura A2 names

php run-g5.php cura A2 muller2csv 
WRONG USAGE - muller2csv needs one parameter. Can be :
  update : modify cura file
  echo : echo detailed names
ex : php run-g5.php cura A2 muller2csv update

php run-g5.php cura A2 muller2csv update
