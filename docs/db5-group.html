Syntax

Group syntax is given by YAML template src/model/Group.yml Function g5\model\Group::newEmpty() uses this template to initialize a group.

Group uid

Group uids are built from the path of the file containing its definition :
data/7-full/group/datasets/newalch/muller402.yml
have uid group/datasets/newalch/muller402

Group id

A group id is unique within groups ; ex muller402

7-full/tmp/group/

Contains text files with one person uid per line
ex: 7-full/tmp/group/datasets/cura/A2.txt
