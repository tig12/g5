Gauquelin5 from scratch

Configure and start using Gauquelin5 software.
This software has been developed and tested under Linux (Ubuntu 18.4), using php 7.2. A priori, it should also work under Windows and Macintosh.

It is used through the command line.

Installation

Open a terminal and clone the repository on your local machine : 
git clone https://github.com/tig12/gauquelin5
The important files and directories are : 
gauquelin5/
    ├── config.yml.dist
    ├── docs/
    ├── run-gauquelin5.php
    ├── share/
    │   ├── 3-cura-marked/
    │   ├── 5-1955-cura-corrected/
    │   └── data/
    ├── src/
    └── tools/
In the rest of this doc, directory gauquelin5/ is refered as the root directory.

Configuration

Create a file config.yml by copying config.yml.dist : 
cp config.yml.dist config.yml
Edit config.yml and adapt the values.

Usage

php run-gauquelin5
will echo an error message : 
usage : 
    php run-gauquelin5.php <serie> <action>
with :
    <serie> = 'A' or 'A1' or 'A2' or 'A3' or 'A4' or 'A5' or 'A6' or '1955' or 'B' or 'B1' or 'B2' or 'B3' or 'B4' or 'B5' or 'B6' or 'D6' or 'D9a' or 'D9b' or 'D9c' or 'D10' or 'E1' or 'E2' or 'E2a' or 'E2b' or 'E2c' or 'E2d' or 'E2e' or 'E2f' or 'E2g' or 'E3' or 'F1' or 'F2'
    <action> depends on serie
Examples :
    php run-gauquelin5.php A2 raw2exported       # will convert file 1-cura-raw/902gdA2.html to 2-cura-exported/A2.csv
Notes :
    - if serie = A, will compute series A1 to A6
    - if serie = B, will compute series B1 to B6
    - if serie = E2, will compute series E2a to E2g
