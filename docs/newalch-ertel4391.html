Ertel 4391
4384 records

This page deals with the file 3a_sports.txt downloaded from newalchemypress.com.
This file is precious because it brings new informations :
- It contains ids of cura.free.fr, comité para, CSICOP, CFEPP - so it's possible to reconstitute these groups and match the different groups without ambiguity.
- It permits to reconstitute parts of Ertel's work.
- It contains eminence information, with the precise source list permitting to reproduce eminence tests.
- Permits to fix missing names of cura A1.

Warning : the code may contain bugs and I don't understand all the columns, so consider the informations about the file as suppositions untill it has been verified.

Integration to gauquelin5

Generate csv

The first step is to generate the file data/5-tmp/newalch-csv/4391SPO.csv : 
php run.php newalch ertel4391 raw2csv

Ertel 4391 generalities

From newalchemypress.com :
In the last year of his life, Professor Ertel kindly sent his main data-collection, of the 4,391 sports champion.
This file is precious for data restoration because records are associated with their identifiers in the different groups dealing with the mars effect story : Gauquelin LERRCP series, CP, CSICOP, CFEPP. So it can be used to compute original lists or to check other restoration processes.

I also found information about this file in these places :
  • "Raising the Hurdle for the Athletes' Mars Effect: Association Co-Varies With Eminence" by Suibert Ertel, published in Journal of Scientific Exploration. Vol. 2. No. 1. pp. 53-82. 1988, available on scientificexploration.org web site.
    Reffered as [Ertel 88] in the rest of this page.
  • "Is the "Mars Effect" Genuine?" by Paul Kurtz, Jan Willem Nienhuys , Ranjit Sandhu, published in Journal of Scientific Exploration, Vol. 11 , No. 1, pp. 19-39, 1997, available on scientificexploration.org web site.
    Reffered as [KNS 97] in the rest of this page.

An incomplete file

One very strange feature of this file : many records don't contain birth time. Surprising because among the missing dates (~800) are records published by Gauquelin (LERRCP series), so known by Ertel. As noted by newalchemypress.com, birth places, longitudes and latitudes are not given. It may be a truncated or draft or imcomplete version of Ertel's file, or Ertel was maybe using complementary files to perform his computations.
Using only information present in this file does not permit to recompute mars positions, but permits to reproduce eminence curves, as mars sector positions are given.

Approximate correspondances

The information extracted from this file can be matched globally to other information sources, but not exactly.
For example,
  • After parsing, the file contains 4384 records ; 7 records are missing to reach 4391.
  • It contains 2087 records coming from file A1, and file A1 contains 2088 records.

Links to other datasets

php run.php newalch ertel4391 look ids
One of the interest of this file is the presence of columns indicating the ids of the records in other data sets.
The information is contained in the following columns :
  • QUEL : origin of the record
  • NR : id (number) in Ertel's reference
  • G_NR : id in Gauquelin's LERRCP (A1 D6 D10)
  • PARA_NR : id in Comité Para test
  • CFEPNR : id in CFEPP test
  • CSINR : id in CSICOP
  • G55 : Presence in Gauquelin 1955 experience
This lists the number of records associated to the external datasets.

Mars sectors

Concerned columns are
  • MARS contains the sectors when the circle is divided in 36.
  • MA12 contains the sectors when the circle is divided in 12.
  • MA_ importance of the sector in the observation of effects.

Combinig these informations with eminence rank, it should be possible to reproduce Ertel's famous curves of 1988.

The table lists the different values found in the 3 columns.
Interesting because it shows the difference between 12 and 36 sectors systems :
In 36-sectors system, sector 9 and 36 are also considered as important, and they are outside sectors 1 and 4 of 12-sectors system.
This shows that 12-sectors system does not catch all the information about the observed statistical effect.
This argument is used to say that using 36-sectors system is more efficient to observe the effects.
