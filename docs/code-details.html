Convention

The code is packaged ; main package name is g5. Code is PSR-4 (package hierarchy correspond to directory hierarchy, like in java). Top-level package (g5) is located in src/.

The program deals with the notion of data source. Each data source corresponds to a sub-package of g5\transform.
Each data source must have a class named Actions. For example, package g5\transform\cura must have a class g5\transform\cura\Actions. This class must implement interface g5\Datasource.
