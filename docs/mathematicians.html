Ranking mathematicians

Learn to buil eminence ranking using mathematicians. See the different indicators available and how to code them, what rules or constants need to be decided by persons knowing the discipline.
The purpose is to build a sorted list of mathematicians that can be used to test eminence effect.
Final sorting results from a global matching..
Global matching is the result of a synthesis of many indicators.
Each indicator has its own logic.
Indicators can be related to each other : relative importance (respective weight)
