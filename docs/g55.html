Gauquelin 1955

Following documentation corresponds to an earlier stage of development, before merging all imported files in a database.
Gauquelin 1955 groups are not yet included in g5 database.
This page deals with data listed in Michel Gauquelin's book "Les hommes et les astres", published in 1955.
Gauquelin announces 5 824 persons.

For each groups, the purpose is to generate two lists :
  • One original list, conform with the list published by Gauquelin.
  • One corrected list.
Related files are located in data/db/init/lerrcp-marked, data/db/init/g55-edited
