This page summarizes the transformations done on files. The process is illustrated using the reconstitution of the group "570 sportifs" from file A1.
The particularity of this process is that it mixes transformations done by programs and by humans.
For the program's point of view, things are quite basic : it just takes file(s) in input, operates computations, and generates other files. The program doesn't have the notions of data source, intermediate files, downloadable files. This knowledge is folded in the structure of directory data/.
To organize this directory, these rules were adopted :
  • Each transformation goes from one directory to an other ; different states of a file are not mixed in a single directory.
  • Very important : Never ever generate a file in a directory containing files edited by humans.
    This permits to avoid accidental destruction of human corrections. When a generated file needs to be edited, it must be copied by a human in a directory containing only edited files.
  • In edited files, only use the edited fields. For example, to generate data in 9-1955-corrected, step edited2corrected() takes the fields that were not modified in 5-1955-edited from 2-cura-csv.
    This permits to have new versions of generated files without needing to update the edited files to integrate the updates.
    Files edited by human are copied once for all, and not modified anymore by a program.
The following diagram illustrates this process :
  1. Store cura.free.fr data on a local machine
    See 1 - Cura data.
  2. Extract data from html pages to generate csv files.
    This step also applies corrections.
    See 2 - Cura to csv
  3. Edit the csv files to associate cura data to Gauquelin 1955 original groups.
    File 2-cura-csv/A1.csv is copied by a human to 3-cura-marked/A1.csv.
    This file id edited ; see 3 - Mark data.
  4. Generate one file per original 1955 group (by program).
  5. Copy these files to directory 5-1955-edited/ (by human).
  6. Correct the files located in 5-1955-edited/ (by human).
    See 4 - Corrections from Gauquelin 1955
  7. Generate downloadable file (by program)
    See 9 - results
