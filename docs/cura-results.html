File D6

  • Restoration of birth cities with reverse geocoding is not satisfying due to the lack of precision of longitudes and latitudes.

File D10

  • Association of birth cities to geonames.org id needs to be done. Maybe possible by program as birth cities are given. This is a bit problematic because cura file contains approximate spelling of city names, but use of approximate string matching (like Levenstein) may be feasible.

Files E1 E3

  • Misses association of birth cities to geonames.org id needs to be done. The problem is similar to D10.
  • Precise given and family names can be tried even for strings that contain more than one whitespace, using the fact that given names are uppercased in cura file.
