Historical tests

Here is a list of the statistical tests done on astrology, possibly incomplete.
The purpose is to cover 1955 - 1996 (Gauquelin, skeptics, Ertel and Müller).
To browse and download the groups that have been restored, see opengauquelin.org/history.
