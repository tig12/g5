402 italian writers
(Arno Müller vol. 1)

This page deals with a test done by Arno Müller in 1991, to replicate an experiment published by the Gauquelins in 1960 on writers and journalists.
The data are composed by
  • The main list of 402 with birth times,
  • A list of 100 without birth time.
This file shares 200 records with Gauquelin data: 197 in A6, 1 in A1, 1 in A2, 1 in A4
It then brings 202 new birth times.
Files generated by g5 can be downloaded from g5.tig12.net/output/history/1991-muller-writers.

Origin

Data for this test were published in Müller's Astro-Forschungs-Daten (AFD) volume 1.
Both Gauquelin and Müller used a book called "Chi è?" (1948 edition) to find famous people from Italy.

Müller's work is made available in electronic format via newalchemypress.com :
  • Page research 5 provides file 5muller_writers.xlsx ; 402 italian writers with birth time and 31 with only birth day (these 31 are part of the second list of 100).
  • Scans of the original document, also sent by newalchemypress.com, permit to build 2 supplementary lists :
    • 100 supplementary writers with birth day but without birth time.
    • A list of 22 writers for which Müller noted a difference with Gauquelin data.
    Scans of the original documents can be downloaded from github.com/tig12/g5-aux/tree/master/1991-muller1-writers

Publications

g5 integration

To include the group of 402 records with birth time : 
php run-g5.php muller m1writers raw2tmp
php run-g5.php muller m1writers tweak2tmp
php run-g5.php muller m1writers gauq update
php run-g5.php muller m1writers tmp2db small
To include the group of 100 records without birth time : 
php run-g5.php muller m1writers raw2tmp100
php run-g5.php muller m1writers tmp2db100 small
Record numbering in Müller's booklet is unique within both lists (402 and 100), which permitted to build the records' Müller unique id without ambiguity.

raw2tmp

Includes several fixes :
  • Some OCR (image to text conversion) errors could be corrected, comparing data with the scans of the original document. These fixes can be seen in the code of class g5\commands\newalch\muller402\raw2tmp
  • Province code (C2) changed to VR for Verona (instead of VE) for 5 records ; the error seems to come from Müller.

Müller 402 and Gauquelin data

A6 writers

Cura file A6 contains 194 italian writers (field PRO = WR); one writer (NUM 31, Guillaume Apollinaire) is present in A6 but not in M402.
Considering only writers, this makes 193 common records between A6 and M402 (in A6, records with NUM from 814 to 1006).

A6 journalists

Modifying the import program to consider also journalists of A6 shows that 4 supplementary records seem to match Müller's writers :
Müller idPersonGauquelin id
1 1898-05-22 Acito Alfredo A6-1354
150 1900-07-25 Dall'Oglio Enrico A6-1489
235 1880-02-14 Govi Mario A6-1568
242 1905-03-09 Guandalini Ugo A6-1578
The matching is not certain because names are missing in A6 file - but records share same birth day and birth place.
They were included, but should be checked, as this may introduce errors in data.

With journalists, this makes 193 + 4 = 197 common records with A6.

Other Gauquelin files

Database import shows that Müller 402 contains 3 other records from Gauquelin data :
Müller idPersonGauquelin id
27 bacci-baccio-maria-1888-01-08 A4-1142
42 beonio-brocchieri-vittorio-1902-05-04 A1-268
443 tallarico-giuseppe-1880-04-28 A2-1595
These matches are included in the code of class newalch\muller402\gauq.
Müller 402 then brings : 402 - 197 - 3 = 202 new birth times.

MUID    FNAME	 GNAME	SEX	OCCU  DATE         TZO	   PLACE	C2	CY	LG      LAT    OCCU	OPUS  LEN
5       Alessi	 Rino	M	WRI   1885-04-30   -0.83   Cervia	RA	IT	012E15  44N24  2   	12    06

Fields

Astro-Forschungs-Daten contains explanations which permit to understand the meaning of the 3 last columns.

OCCU - Profession code

1commediografo
2scrittore
3scrittore combined with other professions
4poeta
5sonstige, ie something other

OPUS - Opus number

Anzahl der veroffentlichen werke - "opus number" - number of published works.

LEN - Article length

Article length in source, in cm.
(Very interesting, permits to rank the writers, usable for building an eminence indicator.)
