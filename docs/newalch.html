Newalchemypress data

Newalchemypress.com web site contains several data sets that complement cura.free.fr. The integration of these data to gauquelin5 program is for the moment limited to name restoration in files A1 and A2.
The files containing lists of famous persons with their names are :
  • Ertel 4391 : a file containing 4384 sportsmen of various origin (Gauquelin, skeptics), used by Suitbert Ertel for his eminence tests.
    Used in gauquelin5 program to restore names in file A1.
  • Müller 1083 : a file containing 1083 physicists for a replication of Gauquelin effects, by Arno Müller and Suitbert Ertel in 1994.
    Used in gauquelin5 program to restore names in file A2.
  • CFEPP 1120 and CFEPP 1066, files containing data from CFEPP (french skeptics) experiment of 1997.
    Not used yet in gauquelin5 program.
  • Müller 402, a file containing italian writers, for a test done by Arno Müller and Françoise Gauquelin in 1993.
    Not used yet in gauquelin5 program.
  • Extracts of "The jupiter temperament and actors" contains associations between actor names and character traits. Related to file A5.
    Not used (yet ?) in gauquelin5 program (seems difficult).
  • 3 files containing A5 writers split in 3 groups : poets, novelists, others.
    Not used yet in gauquelin5 program.
  • Two files with association between persons present in cura files and character traits (one file with 4353 lines, one with 2484).
    Not used yet in gauquelin5 program.
  • Two files containing a restoration of Gauquelin 1955 groups, 570SPO and 576MED.
    Not used yet in gauquelin5 program.
