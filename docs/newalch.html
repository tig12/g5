Newalchemypress data

Newalchemypress.com web site contains several data sets that complement cura.free.fr. The integration of these data to gauquelin5 program is for the moment limited to name restoration in files A1 and A2.
This table lists the files containing birth dates of famous persons with their names found on newalchemypress.com.
Originial
file		g5 integrationDescription
3a_sports.txt Ertel 4391 A file containing 4384 sportsmen of various origin (Gauquelin, skeptics), used by Suitbert Ertel for his eminence tests.
Used in gauquelin5 program to restore names in file A1.
5a_muller_medics.txt Müller 1083 A file containing 1083 physicists for a replication of Gauquelin effects, by Arno Müller and Suitbert Ertel in 1994.
Used in gauquelin5 program to better names in files A2 and E1.

Exploitation of this file by g5 program seems difficult.
