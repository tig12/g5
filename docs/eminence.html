Building eminence ranking

The question is : how to sort the lists of professionals by eminence ?
More precisely : divide the lists in sub-groups of eminence rank.
How to reach a concensus on this process ?
There are many ways to obtain indications about the eminence of a person :
  • Some seem to be purely mechanical, "objective", like Ertel's citation count.
  • Eminence ranking can also be done by humans, producing "subjective" eminence ranking. For example a specialist of a sport can produce a ranking, say "these persons are the top of this disciplin - and here are the good but not outstanding".
In both cases, human intelligence is necessary to rank a person. Ranking is a subjective process, obtaining a consensus is then a delicate question.
This question is not solved here ; this page contains considerations about ranking.

Eminence indicators

The problem is to take a group of persons and associate each of them to an eminence rank. The number of possible distinct ranks is an arbitrary number that must be choosen (Ertel uses 5). In this page, it will be called NRanks

To do this, there are multiple informations at disposition : citation counts, presence in lists, medals, membership...
In this page, each information is called an eminence indicator.

Each professional group is a particular case

A difficulty comes from the fact that each discipline (professional group) has its own sources of information.
For example, for sportsmen, eminence indicators can come from Olympic Games or world or continental or national championships results, or from the number of selections in international competitions. The precise list of indicators varies from sport to sport. Each sport has its own reference competitions, its own reference books or web sites that can be used as indicators. For example, the "Baseball Hall of Fame" is probably a source to use to ranks baseball players, but has no meaning for other sports.

Divide by periods

An other point to consider is that, for a given discipline, the eminence indicators that can be used vary with time. For example, Nobel prizes are only relevant for persons active since the beginning of the 20th century. In order to compare what is comparable, the history of each discipline must be divided in coherent periods of time. Major events like world wars can probably be used as period boundaries for most of the disciplines.

National bias

When trying to rank persons of a given discipline, care must be taken to avoid national bias. For example, if citation count is based on a list of books or web sites, if a particular country is over-represented, there is a risk to over-rate persons of this country.

Indicators may not be reliable

Wikipedia page about Nobel Prize gives examples of important writers who were not awarded, as less important one recieved the prize for litterature. Political and national bias exist in these kinds of lists.

Objective indicators

An "objective" indicator is an indicator that can be computed. These indicators seem to be objective but are in fact subkective.

There are plenty of information sources that can be used for a particular professional group. So far, I've seen two kinds of objective indicators :
  • Sorting indicators, which permit to build sorted lists of persons, like citation count and derivates.
  • Non-sorting indicators, like lists of awards, medals, prizes. These provide information about eminence, but rules must be established to interpret the presence of a person in a given list.

Examples of sorting indicators

Books

This was done by Suitbert Ertel. Given a list of books, established in advance, count the number of times a person is cited, and use this to build a ranking. Ertel's work could be for example reproduced and extended by Google Books.
Here, the arbitrary part lies in the choice of the list of books.

Link count

This can be done from web sites which contain biographies of persons of given professional groups, like Wikipedia. Eminence indicators can be built counting hypertext links pointing to the pages of the different persons of a group. The principle is the same as in citation counts from books.
Here are two examples that permit to rank mathematicians :
  • Part of the data available on Wikipedia can be retrieved using Wikidata. One of the fields that Wikidata offers is called linkcount, which is the number of pages pointing to a given page ; this can be used to rank persons. See the page about Wikidata for details.
  • The MacTutor History of Mathematics archive is a site containing biographies of mathematicians. This site does not provide a service similar to Wikidata, but it's possible to use classical data mining technics to extract data from the site and build a sorted list. This was done by a program published on github.com/tig12/mactutor-by-links.
Here, the arbitrary part also lies in the choice of the sites retained to build the indicators.

Search engines

A similar work can be done counting the number of times a given person has been searched for, and the number of clics to a page concerning this person. This could be done for example using Google Trends API (Application Programing Interface). Google Trends does not give the absolute numbers of queries concerning a person, but permits to compare the numbers related to two or more persons, which is sufficient to sort the persons.
This method seems to contain less arbitrariness than the previous methods.

Subjective ranking

Any person who knows a subject is able to give a list of most famous people in that occupation. Specialists of a given disciplin (like historians of sport) can provide more precise indications and propose a ranking.
This process is entirely subjective, it depends only on the opinion of the persons who build the lists, and in the choice of the persons used to perform the rankings.

Building a consensus

The problem seems delicate :
  • There is a wide variety of information at disposition. For a given group, a synthesis must be built to summarize and obtain one global ranking.
  • Almost all indicators contain a arbitrary or subjective part.
  • Each indicator is possibly biased.
