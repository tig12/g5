Data model

7-full/ contains these notions : document, source, data, person, group.
  • Data
    Defined as a set of key-value pairs.
    Field name => field value (example from file A2)

    Precise data like that are related to 1 source and participate to build 1 person.
  • Person
    Here, persons are famous people who are studied.
    A person is built from 1 - n documents (sources or acts).
    It can belong to 0 - n groups.
  • Group
    A group is a set of persons, ex :
    sportsmen, painters... linked to 1-n data source
    A1, A2, SI42... linked to 1 datasource.
  • Documents
    • Source
      Information source
      Can be a file from the web, a book, review, result of a program execution...
      To follow the construction of a data, many sources are involved.
      Ex for file A1 :
      • Original documents gathered by Gauquelin
      • Lists established by Gauquelin
      • Book published by Gauquelin in 1970
      • First computerized version
      • Transformations done on the data, from Gauquelin to editorial process of cura.free.fr.
      • Transformations done by g5 to produce downloadable file (and possible bugs).
    • Act
      Acts are scans of the original documents.
      Mainly Birth certificates - actes de naissance, décès, mariages.
      Each act is accompanied by a yaml file containing its transcription, see example

trust

Level of reliability of an information. Integer from 1 to 4
  • 1 - reliable, Birth Certificate (BC)
    - Original document available and verifiable by anyone.
    - Transcription approved by 2 persons or more.
  • 2 - better than nothing, Birth Record (BR)
    (= copy of the BC by an officer - may contain mistakes)
    - Original document available and verifiable by anyone.
    - Transcription approved by 2 persons or more.
  • 3 - to check
    Data à priori serious (cura, newalchemypress, astrodatabank) but containing errors.
    Need to be matched against BC.
  • 4 - the rest
    Data without birth time or grabbed from the web (wikidata, web sites).
Most data handled by g5 are level 3.
