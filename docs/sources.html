Information sources

Here are the web sites and persons who provided the data used to build Gauquelin5 database.
They appear in chronological order (order of integration in the database).

  • Gauquelin book "L'influence des astres", 1955
  • newalchemypress.com

cura.free.fr version 5

Online availability of Gauquelin data on cura.free.fr was the departure point of Gauquelin5 program ; see page about Gauquelin data for details.

Note : in november 2020, a new version of CURA database (version 6) was published, but this database became closed data, then not usable by Gauquelin5. This licence was respected, Gauquelin5 only uses informations coming from version 5 of CURA database.
All links to cura.free.fr contained in this documentation point to the version of CURA web site used by the program, still available online thanks to archive.org.

Two other pages of cura.free.fr with birth dates could be used (published before november 2020): Données de Naissance du CURA 1 (455 Birth Data) and Données de Naissance du CURA 2 (41 Birth Data), not yet included in g5 database.

Gauquelin book "L'influence des astres", 1955

A paper copy of this book can easily be found on the web. See pages about Gauquelin data and Gauquelin 1955 for more details.

Nick Kollerstrom

Besides being the author of newalchemypress.com, Nick Kollerstrom provided help by scanning original documents of Arno Müller (612 famous men and 234 famous women). His questions, remarks, complementary informations and enthusiasm were also a great help to develop this program.

Kenneth Irving

Kenneth Irving sent the file which permitted to include CSICOP test in the database ; without his help, restoration of this group wouldn't have been possible.
