Cura - file D10

This file is almost perfect : it contains name, legal birth time, timezone offset, birth place, so importing it is just a matter of formatting.
For sportsmen, precise sport code is not specified and couldn't be computed.
Page 902gdD10.html also contains a list of charcter traits that is not handled by the program. 
php run-g5.php cura D10 all
is equivalent to 
php run-g5.php cura D10 raw2csv
php run-g5.php cura D10 csv2dl
