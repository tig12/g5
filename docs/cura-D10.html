Cura - file D10

This file is almost perfect : it contains name, legal birth time, timezone offset, birth place, so importing it is just a matter of formatting.
For sportsmen, precise sport code is not specified and couldn't be computed.
Page 902gdD10.html also contains a list of character traits that is not handled by the program.
Accoding to cura.free.fr : D10 contains 1398 data of successful Americans but only 1397 were found in the file.
Importing raw file in database is done with following commands: 
php run-g5.php cura D10 raw2tmp
php run-g5.php cura D10 tmp2db
Step raw2tmp contains only minor fixes.
