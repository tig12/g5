Gauquelin 5

This program operates on C.U.R.A Gauquelin archives version 5.
If you are only interested by data generated by the program, you can directly visit this page.
The purpose is to :
  1. Build a programmer-friendly version of C.U.R.A Gauquelin archives.
    • Correct the errors that can be corrected by program.
    • Use Gauquelin's original book (L'influence des astres, 1955) to improve the data.
    • Express dates in a standard format (iso 8601).
    • Associate locations to a common reference (geonames.org).
    • Use a global list of profession codes among all the files.
  2. Restore the original groups used in the tests done by Gauquelin in 1955.
    Provide the exact historical version used by Gauquelin, and a corrected version.
To do this, I used
  • html pages retrieved from cura.free.fr
  • the original book of Gauquelin, "L'influence des astres", 1955.
This permits limited results : a complete restauration of 1955 historical groups, and a partial correction of 1955 groups and Cura data.
A complete correction of data would need an exhaustive check with civil registries ; this is now possible because French civil registries can be freely consulted online - but still a very long and teddious process.
Other documents would be helpful to add new corrections : the booklets "BIRTH AND PLANETARY DATA GATHERED SINCE 1949", published by Michel and Françoise Gauquelin between 1970 and 1982. They contain information that are not present in cura.free.fr data.

Current results

  • Restoration of one 1955 group, "570 sportifs", from serie A1
  • Isolation of other 1955 groups in serie A2 : "576 membres associés et correspondants de l'académie de médecine", "508 autres médecins notables", "349 membres, associés et correspondants de l'académie des sciences"
  • Generation of partially corrected version of CURA files for series A, D, E1, E3, F.

Status


Operates on data retrieved on 2017-04-26
Code developed with php 7.2 ; tested on linux (ubuntu 18.4)
