Gauquelin5
( g5 )

This program operates on data used to perform statistical tests on astrology. These tests, initiated in the 1950s by Michel Gauquelin, followed 3 main directions : tests on professional groups, tests on heredity, and tests on character traits. This program deals with the data used to build professional groups.
The assertions of Gauquelin and followers are :
  • we can observe anomalies in the repartition of some planets in the births of famous people ;
  • the anomalies are more important for very eminent people.
One particular effect (the repartition of Mars in the births of sportsmen) gave rise to very long polemics without any consensual conclusion.

The purpose of Gauquelin5 program is to build new groups in order to reproduce the tests :
  • gather the historical data, check them, correct them, connect them to other datasets ;
  • use the historical datasets to build new groups, corrected and more exhaustive ;
  • Build new eminence indicators and sort famous persons by eminence rank, to obtain ranked data that can make consensus.

Current code is far from these goals ; it contains steps to gather some historical data and correct them. Written code needs tests and debug, this is a work in progress.
It currently uses the following data sources :
  • Data gathered by Michel et Françoise Gauquelin (serie A, files D6, D10, E1, E3), from cura.free.fr C.U.R.A Gauquelin archives version 5 ; N = 15 932
  • Suitbert Ertel and Arno Müller data, from newalchemypress.com
  • Gauquelin 1955 (paper) book "Les hommes et les astres"

You can download generated data, partially restored.

Why Gauquelin5 ?

There are several reasons to care about the data used to perform statistical tests on astrology :
  • Scientific interest
    The "mars effect" polemic lasted for several decades and ended with the Ertel versus Nienhuys arguments, who reached radically opposed opinions. The question to know if these statistical anomalies exist out of human bias is still open. Data collected by Gauquelin and others can serve as a basis to build new groups and execute new tests and see if a consensus can emerge.
  • Historical interest
    Gauquelin and related data are the main material used to perform the astrological tests. So anyone who wants to study the details of this aspect of recent history needs the data. As this is a highly polemic topic, the data permit to check the claims of the historical protagonists.
  • Technical interest
    These data are interesting because they contain birth time. And birth times are generally neglected in the semantic web. Take for example wikidata.org : birth dates are set to 00:00:00. There is no way to make the difference between "this is a date with no time information" and "this is a date with time = 00:00:00" (maybe there is a way, but I didn't find it).
    Gauquelin data permit to better other data sets.

Current implementation

This software is globally still at an immature stage. It brings corrections to Cura an Newalchemypress data but does not produce yet reliable data (legal time, timezone offset and place identification need test and debug).
A log system is missing, to be able to track individual transformations done to each record.
Most of the code concerns data restoration ; preparatory code exists for new group building and eminence ranking.

About this program

The name "Gauquelin5" comes from the main source of information, "Archives Gauquelin version 5", from cura.free.fr.

Code developed with php 7.2 ; tested on linux (ubuntu 18.4)

This documentation is written in a globish as close as possible to english, but uses french conventions for capitalization, spaces and large numbers :
  • Names of months, weekdays, languages etc. are not capitalized.
  • The characters ; : ? ! are preceeded and followed by a white space.
  • White space is used as separator for large numbers (for example one million is written 1 000 000 instead of 1,000,000).

This software has a slow development pace because I do that on my spare time.
