Gauquelin 5

This program operates on data used to perform statistical tests on astrology since 1955.
Raw input data mainly come from C.U.R.A Gauquelin archives version 5 and newalchemypress.com.
The purpose is to :
  • provide corrected versions of existing datasets,
  • restore the groups used in the historical tests when possible,
  • link Gauquelin data with other datasets, like wikidata.org.
Current code provides steps to do that, but restoration process is far form being achieved, see results page.
Current code only deals with data of famous people (serie A, files D6, D10, E1, E3).

If you are only interested by data generated by the program, you can directly go to this page.

Why Gauquelin5 ?

There are several reasons to care about the data used to perform statistical tests on astrology :
  • Scientific interest
    The "mars effect" polemic lasted for several decades and is not finished. It ended with the Ertel vs Nienhuys arguments, and they reached radically opposed opinions. The question to know if these statistical anomalies exist out of human bias is still open.
    Data collected by Gauquelin and others can serve as a basis to execute new tests and see if a consensus can emerge.
  • Historical interest
    Gauquelin and related data are the main material used to perform the astrological tests. So anyone who wants to study the details of this aspect of recent history needs the data.
    As this is a highly polemic topic, the data permit to check the affirmations of the historical protagonists.
  • Technical interest
    These data are interesting because they contain birth time. And birth times are generally neglected in the semantic web.
    Take for example wikidata.org : birth dates are set to 00:00:00. There is no way to make the difference between "this is a date with no time information" and "this is a date with time = 00:00:00" (maybe there is a way, but I didn't find it).
    Storing birth times also obliges to find a way to do it correctly. This problem is detailed in the page about time.
    Introducing birth times in the semantic web is not only useful for astrology ; it can be interesting for demography, sociology etc.

Corrected version of cura files

C.U.R.A. Gauquelin archives published on cura.free.fr have the merit to exist, but before being exploitable, they need to be extracted from web pages and corrected. So the purpose is to build a ready-to-use, user-friendly version of these data.

1 - Generate csv files with a standard format.
  • Express dates in a standard format (iso 8601).
  • Use a common list of profession codes among all the generated files.
  • Associate locations to a common reference (geonames.org).
  • Express countries in a standard format (iso 3166).
  • Restore precise family names and given names when possible.

2 - Correct the errors that can be corrected.
A complete correction of data would need an exhaustive check with civil registries ; this is now possible for persons born before world war I, because French civil registries can be freely consulted online - but it's a very long and teddious process and hasn't been done.

Restoration of the original data sets

Ideally, a restoration process would lead to two versions of the data sets used in the historical tests : an exact historical version used by actors (errors included) and a corrected version.
The program, in its current state, only operates on the groups published in 1955 by Michel Gauquelin.
Restoration of the exact historical version has only been done for one group (570 sportifs), using the original book of Gauquelin, "L'influence des astres", published in 1955.

Current results

  • Generation of partially corrected version of CURA files for serie A and files D6, D10, E1, E3.
  • Restoration of one 1955 group, "570 sportifs", from serie A1
  • Isolation of other 1955 groups in serie A2 : "576 membres associés et correspondants de l'académie de médecine", "508 autres médecins notables", "349 membres, associés et correspondants de l'académie des sciences"
