Gauquelin5

The primary purpose of this program is to restore the original groups used by Michel Gauquelin since 1955 to perform statistical tests on astrology. This is done using "Archives Gauquelin" version 5, available on cura.free.fr web site.
This code was used to reconstitute the group of "570 sportifs", which was at the beginning of the "mars effect" polemics.
This code can also be used to generate a corrected version of cura.free.fr data.
If you are not interested by the details of the code, you can directly dowload the results.

Usage

  • Copy the html pages containing the data on your local machine (you can use for example script tools/get-data).
  • Copy config.yml.dist to config.yml and adapt the values of source-dir and ouptut-dir.
  • In a shell console, go to the root directory of the program (containing the file run-gauquelin5.php).
    General syntax is : 
    php run-gauquelin5.php <serie> <action>
    If you run : 
    php run-gauquelin5.php
    it displays a message with possible values for <serie>.

    If you run for example : 
    php run-gauquelin5.php A1
    it displays a message with possible actions for serie A1.

    For example, to convert cura html files to csv and apply corrections: 
    php run-gauquelin5.php A1 raw2exported
    it extracts the data from A1 html file, applies some corrections and generates a csv file.
