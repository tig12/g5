Gauquelin 5

This program operates on data used to perform statistical tests on astrology since 1955.
Raw input data mainly come from C.U.R.A Gauquelin archives version 5 and newalchemypress.com.
The purpose is to :
  • provide corrected versions of existing datasets,
  • restore the groups used in the historical tests when possible,
  • link Gauquelin data with other datasets, like wikidata.org.
Current code only deals with data of famous people (serie A, files D6, D10, E1, E3).
Current code provides steps to do that, but restoration process is far form being achieved, see results page.

If you are only interested by data generated by the program, you can directly go to this page.

Corrected version of cura files

C.U.R.A. Gauquelin archives published on cura.free.fr have the merit to exist, but before being exploitable, they need to be extracted from web pages and corrected. So the purpose is to build a ready-to-use, user-friendly version of these data.

1 - Generate csv files with a standard format.
  • Express dates in a standard format (iso 8601).
  • Use a common list of profession codes among all the generated files.
  • Associate locations to a common reference (geonames.org).
  • Express countries in a standard format (iso 3166).
  • Restore precise family names and given names when possible.

2 - Correct the errors that can be corrected.

Restoration of the original groups

The purpose is to restore the groups used in the historical tests ; provide the exact historical version used by actors (errors included) and a corrected version.
To do this, I used :
  • html pages retrieved from cura.free.fr,
  • the original book of Gauquelin, "L'influence des astres", 1955.
In theory, this permits a complete restoration of 1955 original groups and a partial correction. In practice, this is partially done.

Restoration

The way this step has been done implies human corrections :
  • Associate records of cura files to the 1955 groups.
  • Check each record to note the differences between 1955 book and cura file. This step is long and involves 2 persons : one reads the book, the other types the differences.

Correction

A complete correction of data would need an exhaustive check with civil registries ; this is now possible because French civil registries can be freely consulted online - but still a very long and teddious process and hasn't been done. Instead, the corrected versions of 1955 groups are built using a mix of human corrections (for names, names of birth places) and cura data (for dates). This relies on the hypothesis that cura data contain less errors that Gauquelin 1955 book.

Other documents would be helpful to add new corrections, in particular the booklets "BIRTH AND PLANETARY DATA GATHERED SINCE 1949", published by Michel and Françoise Gauquelin (LERRCP) between 1970 and 1982. They contain information that are not present in cura.free.fr data.

Current results

For the moment, work has been done only on lists containing names.
  • Generation of partially corrected version of CURA files for series A, D6, D10, E1, E3.
  • Restoration of one 1955 group, "570 sportifs", from serie A1
  • Isolation of other 1955 groups in serie A2 : "576 membres associés et correspondants de l'académie de médecine", "508 autres médecins notables", "349 membres, associés et correspondants de l'académie des sciences"
