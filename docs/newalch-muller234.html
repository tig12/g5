Status: under development - integration to g5 incomplete.

Origin

This list was published in Müller's Astro-Forschungs-Daten (AFD) volume 3.

Publication of this list is possible thanks to Nick Kollerstrom, who scanned the original booklet.
Scans of the list + field meanings can be downloaded from g5.tig12.net/auxiliary/1993-muller-women.

These scans were transformed to a text file used as input by g5 : data/raw/muller/afd3-women/muller-afd3-women.txt
